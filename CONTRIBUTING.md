# Contributing to Aura

Thank you for your interest in contributing to Aura! We welcome contributions from the community to help make this AI audio platform even better.

### 1. Fork the Repository
Click the "Fork" button on the top right of the repository page.

### 2. Clone your Fork
```bash
git clone https://github.com/Start a new browser session./aura.git
cd aura
```

### 3. Create a Feature Branch
```bash
git checkout -b feature/AmazingFeature
```

### 4. Setup Development Environment
Please refer to the [Root README](./README.md) for detailed setup instructions for both `aura-ai` (Frontend) and `aura-ml` (Backend).

### 5. Make Changes & Test
- Ensure you run existing tests.
- Add new tests for your features if applicable.

### 6. Commit Changes
```bash
git commit -m "Add some AmazingFeature"
```

### 7. Push to the Branch
```bash
git push origin feature/AmazingFeature
```

### 8. Open a Pull Request
Go to the original repository and click "New Pull Request".
